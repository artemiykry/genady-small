# IncidentList

All incidents, filterable by status.

- **Purpose** — see what's open or recently closed.
- **Shows** — incident rows with title, status badge, owner, started_at, time since resolved.
- **Actions** — `create_incident`; filter by status; open `IncidentPage`.
- **States**
  - *all* — every status.
  - *open* — only `open` and `postmortem`.
  - *closed* — only closed (archive view).
- **Composed of** — incident row.
