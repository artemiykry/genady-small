# NoteList

All notes, recent first.

- **Purpose** — find or revisit captured thoughts.
- **Shows** — note rows with title (or first line), updated_at, archived badge.
- **Actions** — `create_note`; open `NotePage`; filter active / archived.
- **States**
  - *active* — non-archived.
  - *archived* — archived only.
- **Composed of** — note row.
